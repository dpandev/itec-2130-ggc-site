HTML elements are of two types.
  1. 1. Paired tags
  2. 2. NonPaired Tags(Empty tags)

Paired Tags

  • It has open and close tags
  • li, ul, ol, body, head, html, title, h2, h1, h3

NonPaired Tags

  • It is only a open tag.
  • Example: <br>, meta, !DOCTYPE
Heading Tags h1 to h6
Heading Tags are used to create headings and make the contents automatically bold and big.
